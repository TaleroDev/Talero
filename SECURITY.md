# Security Policy

## Supported Versions

This repository is currently in active setup.

| Version | Supported |
| --- | --- |
| `main` | yes |
| older commits/tags | no |

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report privately to:

- GitHub Security Advisories (preferred): use the "Report a vulnerability" button in the repository Security tab
- Email fallback: `security@talero.info`

Please include:

- A clear description of the issue
- Reproduction steps / proof of concept
- Estimated impact
- A suggested fix if available

## Disclosure Process

- Initial acknowledgement target: within 72 hours
- Triage and severity assessment: as fast as possible
- Fix timeline: depends on severity and operational risk
- Public disclosure: after a fix is available and users have had time to update

We appreciate responsible disclosure and coordinated release timing.
